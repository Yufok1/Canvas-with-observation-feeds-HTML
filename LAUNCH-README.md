# 🚀 Canvas Launch Script

## Overview
`launch-canvas.bat` and `launch-canvas-fixed.bat` are comprehensive Windows batch scripts that provide a god-tier ASCII art interface for launching the Canvas with Live AI Observation Feeds system.

## Available Scripts

### `launch-canvas.bat` (Original)
- Features pipe characters (`|`) for visual separation
- May have compatibility issues with some Windows terminals
- More visually structured layout

### `launch-canvas-fixed.bat` (Recommended)
- Uses dash characters (`-`) for better compatibility
- Guaranteed to work on all Windows systems
- Same functionality with improved reliability

## Features
- 🎨 **Beautiful ASCII Art Interface**: God-tier visual design showcasing the AI orchestration matrix
- 🤖 **System Architecture Display**: Detailed information about all AI systems and their roles
- ⚡ **Automated Setup**: One-click server startup and browser launch
- 📋 **Status Monitoring**: Real-time initialization progress
- 🛡️ **Requirements Check**: Clear system requirements display

## Usage
1. **Double-click** either batch file or run it from command prompt
2. **Watch the initialization sequence** with beautiful ASCII art
3. **Browser opens automatically** to the canvas
4. **AI systems activate** and begin monitoring

## What It Does
1. **Displays System Emblem**: Shows the CANVAS branding and AI orchestration matrix
2. **Lists AI Systems**: Details all 5 AI entities and their functions
3. **Shows Architecture**: Displays system components and capabilities
4. **Starts HTTP Server**: Launches Python HTTP server on port 8000
5. **Opens Browser**: Automatically navigates to the canvas
6. **Provides Status**: Shows initialization progress and next steps

## ASCII Art Design
The script features:
- **Unicode Emojis**: 🤖🧠🔮🌊🐋🔱 for visual appeal
- **Box Drawing Characters**: Professional borders and layouts
- **Color-coded Sections**: Different areas for different information
- **Progressive Loading**: Step-by-step initialization display
- **System Status**: Real-time progress indicators

## Requirements
- Windows OS
- Python installed and in PATH
- Modern terminal that supports Unicode
- Ollama running (for AI functionality)

## Quick Start
```batch
# Use the fixed version for best compatibility:
launch-canvas-fixed.bat

# Or use the original version:
launch-canvas.bat
```

The script will handle everything automatically and provide a cinematic launch experience worthy of the AI orchestration system!
